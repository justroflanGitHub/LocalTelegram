## Getting Started

### Registration

1. **Download the App**
   - Windows: Download installer from your administrator
   - Android: Download APK from your administrator

2. **Create Account**
   - Open the app
   - Tap "Register"
   - Enter your email address
   - Create a strong password
   - Enter the invite code provided by your administrator

3. **Verify Email**
   - Check your email for verification code
   - Enter the code in the app

4. **Complete Profile**
   - Add your name
   - Optionally add profile photo
   - Set your username

### First Login

1. Enter your email and password
2. If 2FA is enabled, enter the code from your authenticator app
3. Grant necessary permissions (notifications, microphone, camera)

## Account & Profile

### Viewing Your Profile

1. Tap your avatar in the top-left corner
2. View your profile information:
   - Name
   - Username
   - Email
   - Phone (if added)
   - Bio

### Editing Profile

1. Go to Settings → Profile
2. Tap "Edit Profile"
3. Change:
   - First name
   - Last name
   - Username
   - Bio (up to 200 characters)
   - Profile photo

### Changing Profile Photo

1. Go to Settings → Profile
2. Tap on your current photo
3. Choose:
   - Take Photo (camera)
   - Choose from Gallery
   - Remove Photo

### Changing Password

1. Go to Settings → Security
2. Tap "Change Password"
3. Enter current password
4. Enter new password (minimum 8 characters)
5. Confirm new password

### Account Deletion

⚠️ **Warning**: This action is permanent and cannot be undone.

1. Go to Settings → Account
2. Tap "Delete Account"
3. Enter your password to confirm
4. All your data will be deleted within 30 days

## Messaging

### Starting a New Chat

1. Tap the "+" button (or "New Chat")
2. Select a contact from your list
3. Or search by name/username
4. Start typing your message

### Sending Messages

**Text Messages:**
1. Type in the message field
2. Tap Send (➤) or press Enter

**Formatting:**
- **Bold**: `*text*`
- *Italic*: `_text_`
- `Monospace`: ``` `text` ```
- [Link]: `[text](url)`

### Message Actions

**Reply to Message:**
1. Hover over message (or long-press on mobile)
2. Select "Reply"
3. Type your reply
4. Send

**Edit Message:**
1. Hover over your message
2. Select "Edit"
3. Make changes
4. Save

**Delete Message:**
1. Hover over message
2. Select "Delete"
3. Confirm deletion

**Forward Message:**
1. Select message(s)
2. Tap "Forward"
3. Choose recipient(s)
4. Send

### Reactions

1. Hover over message
2. Click the reaction icon (😊)
3. Select an emoji reaction
4. Reaction appears below the message

### Pinned Messages

**Pin a Message:**
1. Select message
2. Choose "Pin"
3. Confirm pinning

**View Pinned Messages:**
- Pinned messages appear at the top of the chat
- Click to expand and view all pinned messages

### Message Search

1. Open a chat
2. Tap the search icon
3. Enter search term
4. Results show messages containing the term

### Message Status

| Icon | Status |
|------|--------|
| ○ | Sending |
| ✓ | Sent |
| ✓✓ | Delivered |
| ✓✓ (blue) | Read |

## Groups

### Creating a Group

1. Tap "New Group" from the menu
2. Enter group name
3. Add group photo (optional)
4. Select members from your contacts
5. Tap "Create"

### Group Settings

**As Owner/Admin, you can:**
- Change group name and photo
- Add/remove members
- Promote/demote admins
- Set group description
- Create invite links
- Delete group

### Group Roles

| Role | Permissions |
|------|-------------|
| Owner | All permissions, can delete group |
| Admin | Add/remove members, edit info, pin messages |
| Member | Send messages, view members |

### Joining a Group

**Via Invite Link:**
1. Click the invite link
2. Preview group info
3. Tap "Join Group"

**Via Direct Invite:**
1. Receive notification
2. Tap to view invite
3. Accept or Decline

### Leaving a Group

1. Open group chat
2. Tap group name (header)
3. Scroll down
4. Tap "Leave Group"
5. Confirm

### Group Invite Links

**Create Link:**
1. Open group settings
2. Tap "Invite via Link"
3. Copy or share link

**Revoke Link:**
1. Open group settings
2. Tap "Invite via Link"
3. Tap "Revoke Link"
4. New link will be generated

## Files & Media

### Sending Files

**From Computer:**
1. Tap attachment icon (📎)
2. Select "File"
3. Browse and select file
4. Add caption (optional)
5. Send

**From Mobile:**
1. Tap attachment icon
2. Choose:
   - Photo/Video (from gallery)
   - File (from storage)
   - Camera (take photo/video)

### File Size Limits

| Type | Maximum Size |
|------|--------------|
| Photos | 50 MB |
| Videos | 2 GB |
| Documents | 2 GB |
| Voice Messages | 100 MB |

### Voice Messages

**Recording:**
1. Tap and hold microphone icon
2. Speak your message
3. Release to send
4. Or slide up to lock recording

**Listening:**
1. Tap play button on voice message
2. Adjust speed (0.5x - 2x)
3. Pause/resume as needed

### Photo/Video Gallery

**Viewing Media:**
1. Open chat
2. Tap chat name
3. Tap "Media"
4. Browse all shared media

**Saving Media:**
1. Open photo/video
2. Tap download/save icon
3. Saved to device

## Video Calls

### Starting a Call

**Individual Call:**
1. Open chat with contact
2. Tap video call icon (📹) or voice call icon (📞)
3. Wait for answer

**Group Call:**
1. Open group chat
2. Tap video call icon
3. Call starts, members can join

### During a Call

**Controls:**
- 🎤 Mute/Unmute microphone
- 📹 Enable/Disable camera
- 📱 Switch camera (front/back)
- 🖥️ Share screen
- 🔊 Speaker on/off
- ❌ End call

**Screen Sharing:**
1. Tap screen share icon
2. Select screen/window to share
3. Share your screen
4. Tap to stop sharing

### Joining a Call

1. Receive call notification
2. Tap "Answer" or "Decline"
3. For group calls, tap "Join" in the chat

### Call Quality

- HD video up to 720p
- Adaptive quality based on network
- Works on 3G/4G/WiFi

## Privacy & Security

### Two-Factor Authentication (2FA)

**Enable 2FA:**
1. Go to Settings → Security
2. Tap "Two-Factor Authentication"
3. Choose method:
   - Authenticator App (recommended)
   - SMS
4. Scan QR code or enter secret
5. Enter verification code
6. Save backup codes securely

**Disable 2FA:**
1. Go to Settings → Security
2. Tap "Two-Factor Authentication"
3. Enter current code
4. Tap "Disable"

### Privacy Settings

**Last Seen:**
1. Go to Settings → Privacy
2. Tap "Last Seen"
3. Choose who can see:
   - Everyone
   - My Contacts
   - Nobody
4. Add exceptions if needed

**Profile Photo:**
1. Go to Settings → Privacy
2. Tap "Profile Photo"
3. Choose visibility

**Phone Number:**
1. Go to Settings → Privacy
2. Tap "Phone Number"
3. Choose visibility

### Blocking Users

**Block a User:**
1. Open user's profile
2. Tap "Block"
3. Confirm

**Unblock:**
1. Go to Settings → Privacy
2. Tap "Blocked Users"
3. Find user and tap "Unblock"

### Active Sessions

**View Sessions:**
1. Go to Settings → Security
2. Tap "Active Sessions"
3. See all logged-in devices

**Terminate Session:**
1. Find the session
2. Tap "Terminate"
3. Or "Terminate All Other Sessions"

## Settings

### Notifications

**Configure Notifications:**
1. Go to Settings → Notifications
2. Toggle:
   - Message notifications
   - Group notifications
   - Call notifications
   - Sound
   - Vibration
   - Preview

**Per-Chat Settings:**
1. Open chat
2. Tap chat name
3. Tap "Notifications"
4. Choose:
   - Default
   - Mute (1 hour / 8 hours / 2 days / Always)
   - Custom

### Appearance

**Theme:**
1. Go to Settings → Appearance
2. Choose:
   - Light
   - Dark
   - System default

**Text Size:**
1. Go to Settings → Appearance
2. Adjust slider

**Chat Background:**
1. Go to Settings → Appearance
2. Tap "Chat Background"
3. Choose color or image

### Data & Storage

**Auto-Download:**
1. Go to Settings → Data & Storage
2. Configure:
   - Photos
   - Videos
   - Files
   - Voice messages

**Storage Usage:**
1. Go to Settings → Data & Storage
2. Tap "Storage Usage"
3. View usage by chat
4. Clear cache if needed

### Language

1. Go to Settings → Language
2. Select preferred language
3. App restarts in new language

## Troubleshooting

### Can't Log In

1. Check internet connection
2. Verify email and password
3. Check if 2FA is enabled
4. Try "Forgot Password"
5. Contact administrator

### Messages Not Sending

1. Check internet connection
2. Check if you're blocked by recipient
3. Restart the app
4. Clear app cache

### Notifications Not Working

1. Check system notification settings
2. Check app notification settings
3. Disable battery optimization (Android)
4. Reinstall app

### Video Call Issues

1. Check camera/microphone permissions
2. Check internet speed (min 1 Mbps)
3. Close other apps using camera
4. Try audio-only mode

## FAQ

**Q: How do I change my username?**
A: Go to Settings → Profile → Edit Profile → Username

**Q: Can I recover deleted messages?**
A: No, deleted messages cannot be recovered

**Q: How many people can be in a group?**
A: Up to 200 members (configurable by administrator)

**Q: Are my messages encrypted?**
A: Messages are encrypted in transit. Secret chats offer end-to-end encryption.

**Q: How do I report a user?**
A: Open user's profile → Report → Select reason

**Q: Can I use LocalTelegram on multiple devices?**
A: Yes, you can be logged in on multiple devices simultaneously

**Q: How do I export my data?**
A: Go to Settings → Account → Export Data
